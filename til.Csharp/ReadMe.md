##  2. <a name='FilesandFolders'></a>Files and Folders

| name           | content                                                       |
| -              | -                                                             |
| `.vscode`      | vscode debug settings                                         |
| `gitignore`    | files and folders not to include in git version control       |
| `bin\`         | built binaries                                                |
| `obj\`         | built artifacts                                               |
